# Cloud Security & Pentesting

In diesem Ordner befinden sich reflektierte Write-ups, Notizen und Denkmodelle
rund um Cloud-Pentesting.

Der Fokus liegt auf:
- Architekturverständnis
- Berechtigungsmodellen (z. B. IAM, STS)
- Service-Interaktionen
- methodischem Vorgehen (Kill Chains, Denkpfade)

Keine Schritt-für-Schritt-Anleitungen,
sondern Nachbetrachtungen, Analyse und Lernnotizen.
