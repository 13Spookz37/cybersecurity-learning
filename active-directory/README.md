## Methodischer Ansatz

Dieses Repository bildet Active‑Directory‑Pentesting als zusammenhängendes System ab.

Statt linearer Schritt‑für‑Schritt‑Anleitungen werden hier Phasen, Abhängigkeiten
und mögliche Angriffspfade dargestellt. Ziel ist es, ein mentales Modell dafür zu entwickeln, wie sich
Informationen, Berechtigungen und strukturelle AD‑Eigenschaften
über mehrere Phasen hinweg auswirken.

Die Inhalte dienen als methodische Referenz und Denkstruktur – nicht als
ausführbare Anleitung.
